# brandeduk-frontend

Frontend application for BrandedUK e-commerce platform.

## Features

- Product browsing and filtering
- Product customization
- Shopping cart and quote management
- Responsive design for desktop and mobile

## Setup

1. Open `home.html` in a web browser or serve via a local web server
2. Ensure the backend API is running at `https://brandeduk-backend.onrender.com/api`

## Structure

- `home.html` - Main product listing page
- `brandedukv15-child/` - Theme assets (CSS, JS, images)
- `mobile/` - Mobile-specific pages
- `product-detail.html` - Individual product pages
- `customize-positions.html` - Product customization interface

## API Integration

The frontend communicates with the backend API at:
- Base URL: `https://brandeduk-backend.onrender.com/api`
- Products endpoint: `/products`
